# Economic Recovery After Wildfires: Evidence from County-Level Data in California
_This project studies how major wildfire events affect county-level employment and economic activity in California._

# Data sources
This project combines multiple public datasets on wildfire incidents and local economic activity.

## CAL FIRE Wildfire Incident Data
Used to identify major wildfire events and affected counties.

| Data source | Variables used | Purpose |
|-------------|----------------|---------|
| CAL FIRE incident dataset | incident location, county, incident date, acres burned, spatial coordinates | Identify wildfire events and affected counties |

## BLS Quarterly Census of Employment and Wages (QCEW)
Used to track employment changes over time.

| Data source | Variables used | Purpose |
|-------------|----------------|---------|
| QCEW | county name, employment counts, industry categories, quarterly employment levels | Measure county-level employment trends |

## BEA Regional Accounts Data
Used to track county-level GDP and economic output.

| Data source | Variables used | Purpose |
|-------------|----------------|---------|
| BEA regional accounts | county GDP, annual economic output, industry breakdowns | Measure local economic activity |
